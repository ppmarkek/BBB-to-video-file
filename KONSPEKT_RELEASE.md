# Konspekt for Windows

Запуск: дважды кликни по файлу "Konspekt.exe".

Не требуется Python, .venv или .bat. Не перемещай Konspekt.exe отдельно: ему нужна папка _internal рядом с ним.

В выпуск уже включены:

- FFmpeg для извлечения аудио и кадров;
- Tesseract OCR для текста с экрана;
- Faster-Whisper и его локальные зависимости.

При первой обработке Whisper скачает выбранную открытую модель на компьютер. Это не API-вызов и не требует токенов. Интернет также нужен, чтобы загрузить публичную BBB-запись.

Материалы лекций хранятся локально в %LOCALAPPDATA%\Konspekt.
